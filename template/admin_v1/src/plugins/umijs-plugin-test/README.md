[![NPM downloads](http://img.shields.io/npm/dm/umijs-plugin-test.svg?style=flat)](https://npmjs.org/package/umijs-plugin-test)



## Install

```bash
# or yarn
$ npm install
```

```bash
$ npm run build --watch
$ npm run start
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['umijs-plugin-test'],
  ],
}
```
